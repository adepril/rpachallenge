In this challenge, we have to fill out forms with data which is stored in an Excel file.
Each time we submit the form, the order of the fields changes

Both solutions offer the same tools to implement a process.
So I tackled the "RPA Challenge", coding the same process on both platforms.
Now, which platform is the fastest to complete this challenge ?

  • UiPath
  • Automation Anywhere

UiPath

About 57 seconds to complete 10 forms

Workflow

Here is an overview of the script. Each part is detailed below.

1 Test if the Excel file is present on the computer.

2 If not, click on "Download button".

3 Thanks to the Excel activity, we create a DataTable with the content of the sheet.

4 Then we press the button to start the challenge.

In the "For each row" section, we are going to loop through "dtChallenge" DataTable to insert all datas.

At each iteration, we fill in 7 fields.

  • Fist Name
  • Last Name
  • Address
  • Email
  • Phone Number
  • Company Name
  • Role In Company

Each time the location of the fields is different.

The solution is to look for the name of the field, then insert the data in the Input field just below it.

See below for more details, using the "First Name" field as an example.

Finally, when the 7 fields are filled, we click on Submit button. Then we do another iteration to fill another page.

Here, we take the example of the "First Name" field, but the principle is the same for the 7 fields.

We put in a variable (columnName) the name of the Label field we want to reach.

Then we use this variable in the selector of the "Find Element" activity.

Finally, we fill the Input field with this same variable.
